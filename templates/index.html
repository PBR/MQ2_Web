MQ²¶

MapQTL is a powerful tool to do QTL mapping. It allows you to analyze multiple traits very quickly and in a user-friendly way. However, if you perform QTL mapping on a lot of traits, it is hard to get an overview of the results.

MQ² aims at filling this need. Assuming one QTL per linkage group and using the LOD threshold set by the user. It extracts all the QTLs detected by MapQTL and using a map file, it finds the marker closest to the peak and finally calculates the number of QTLs found near each marker on the map. Note that this works for Interval mapping, MQM or RQM mapping but not for KW mapping.

This approach quickly allows you to find and visualize potential QTL hotspot in your dataset. This is particularly useful for large QTL analysis on a large number of traits.

About the name:

The aim of this tool is to visualize QTLs detected using MapQTL.
So, it is MapQTL to QTLs,
QTL * QTL = QTL²
thus MQ².
